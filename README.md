#  Student Management System

##  Overview
This project is a Student Management API built using FastAPI. It includes CRUD operations, database integration, data analysis, machine learning, testing, Docker setup, and CI/CD pipeline.

##  Features
- Add, update, delete, and fetch students
- Search students by name
- Database using SQLite + SQLAlchemy
- Data analysis using Pandas
- Visualization using Matplotlib
- Machine learning model for prediction
- API testing using Pytest
- CI/CD using GitHub Actions

##  Tech Stack
- Python
- FastAPI
- SQLAlchemy
- Pandas
- Scikit-learn
- Pytest
- Docker
- GitHub Actions

##  API Endpoints
- GET /students
- POST /students
- GET /students/{id}
- PUT /students/{id}
- DELETE /students/{id}
- GET /search?name=Ali

##  How to Run
Run the following command:

uvicorn main:app --reload


Open:
http://127.0.0.1:8000/docs
